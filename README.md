## How to View

### GitHub Pages
After enabling GitHub Pages, visit: https://leonardolondono.github.io/OneG-Docs/

### Local Development
1. Clone the repository
2. Open `index.html` in any modern browser
3. Click navigation to explore different sections
4. Move your mouse to interact with the particle animation

### Static Presentation
1. Open `presentation.html` in browser
2. Scroll to navigate between slides

## Technology Stack

### Platform
- **Frontend**: React 19 + TypeScript + Vite 6
- **Backend**: 18 Fastify microservices
- **Database**: PostgreSQL (Supabase) with schema-per-service
- **Workflows**: Temporal
- **Infrastructure**: Docker Compose

### Documentation Site
- Pure HTML/CSS/JavaScript
- No build step required
- CDN-hosted technology logos
- Responsive design

## Enabling GitHub Pages

1. Go to your repository Settings
2. Navigate to Pages section
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click Save
6. Wait a few minutes for deployment
